# Machine Learning Prediction Model

## 📌 Problem Statement
Build a classification model to predict outcomes based on input features.

## ⚙️ Approach
- Used Iris dataset
- Split data into training and testing sets
- Trained a Random Forest model
- Evaluated accuracy

## 🧠 Technologies Used
Python, Scikit-learn

## 📊 Output
Model predicts class labels with good accuracy.

## 🚀 Future Improvements
- Use advanced models like XGBoost
- Deploy model using Flask or Streamlit
